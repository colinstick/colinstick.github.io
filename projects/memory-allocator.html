Overview

Designed and implemented a custom dynamic memory allocator in C, recreating core functionality of malloc and free from scratch. The project focused on low-level systems programming concepts including explicit free lists, memory alignment, block splitting, deferred coalescing, and heap consistency checking. I optimized the allocator for both memory utilization and throughput while debugging complex pointer and heap management behavior using GDB and custom validation tools.

Command Interpreter

Jan 2026

C School

Overview

Developed a command-line interpreter for the ASML programming language in C, building core components including a parser, interpreter, and custom hash table while working extensively with Linux tools and low-level memory management. The project involved implementing arithmetic, bitwise, memory, and control-flow operations, debugging with GDB and Valgrind, and strengthening proficiency in C programming, pointers, data structures, and systems-level development on Linux.
